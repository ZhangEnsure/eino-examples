# Tools — Usage Patterns & Gotchas

## Main Agent Tools

- `read_file`: 读取文件内容，支持指定起始行和行数
- `tree`: 查看目录树结构
- `diary_write`: 向每日日记追加条目（append-only）
- `memory_edit`: 编辑 agent-core 记忆文件（SOUL.md/AGENTS.md/TOOLS.md/MEMORY.md）
- `diary_search`: 在日记目录中搜索关键词

## CodeAgent Tools

- `bash`: 执行 shell 命令
- `edit_file`: 创建或覆盖文件内容
- `python_runner`: 执行 Python 代码
